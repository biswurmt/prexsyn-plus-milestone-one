\section*{Proposed methods}

The benchmarking pipeline seeds four modification methods (CReM, mmpdb, JT-VAE, and LibINVENT) with PrexSyn's top-scoring candidate per specification. Each method generates a set of structural variants, which are screened through a shared synthesizability and property-conservation pipeline alongside an equal number of additional PrexSyn draws from the same specification. This repeated-sampling baseline establishes what PrexSyn alone achieves with more compute, so each modification method is evaluated by how many viable candidates it produces that repeated sampling cannot.

\textit{CReM} \cite{bib7} replaces molecular fragments (side chains, ring systems, functional groups) with ChEMBL database alternatives, subject to chemical validity constraints. A context radius controls the edit scale: radius 1 makes conservative R-group substitutions (e.g., methyl $\rightarrow$ chloro); radius 2 permits ring modifications; radius 3 enables scaffold-level replacements. We benchmark all three radii to characterize the edit-scale/quality tradeoff within a single method.
\textit{mmpdb} \cite{bib8} applies matched molecular pair (MMP) rules: single-site substitutions extracted from pairs of real compounds that differ by exactly one structural change (e.g., methyl $\rightarrow$ chloro, phenyl $\rightarrow$ pyridyl), with empirical property-change annotations enabling property-conserving transformations. These are the most conservative edits in our benchmark.
\textit{JT-VAE} \cite{bib9} encodes molecules into a continuous latent space by decomposing structure into a junction tree (a hierarchy of chemically meaningful substructures: ring systems, bonds, atom clusters). Perturbing the seed's latent vector and decoding enables scaffold hopping: replacing an entire core structure while preserving the pharmacological profile, a reach beyond any fragment-level method.
\textit{LibINVENT} \cite{bib10,bib11} preserves the molecular scaffold (core ring system and connectivity) from PrexSyn's output while generating alternative R-group decorations (the peripheral functional groups that drive selectivity and pharmacokinetics) via reinforcement learning steered toward target properties.

Spanning rule-based (CReM, mmpdb) and learned (JT-VAE, LibINVENT) paradigms across a local-to-global edit-scale axis, these methods were selected from a systematic survey of $\sim$50 candidates to maximize coverage. No prior work benchmarks modification methods as post-generation complements to synthesis-aware generation; complementarity analysis further identifies which combinations maximize total unique-hit coverage, informing pipeline design for medicinal chemistry and automated discovery.
